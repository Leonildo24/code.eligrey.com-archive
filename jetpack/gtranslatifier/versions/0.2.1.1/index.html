GTranslatifier Jetpack feature - Version 0.2.1.1

Translatification at your finger tips.

The GTranslatifier Jetpack feature helps you translate web pages and selected text using Google Translate.

How to use GTranslatifier

To translate a web page into your language, just click on the icon in your status bar. To open the translated web page in a new tab, right click or middle click on the icon. To un-translate a translated page, click on the icon again. To translate a specific portion of a web page, select the text you wish to translate and then click on the icon.

Source code

The source code for the GTranslatifier Jetpack feature can be found at gtranslatifier.js.
